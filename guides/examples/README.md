# Examples

This directory contains real-world examples of AI-ready documentation in action.

## Available Examples

### Example Categories

#### Success Stories
- **Before/After Documentation**: Examples showing transformation from traditional to AI-ready documentation
- **Implementation Cases**: Real projects that successfully used these templates
- **AI Agent Interactions**: Documented interactions between AI agents and documentation

#### Use Case Examples
- **Web Application Feature**: Complete ticket and implementation example
- **API Integration**: Documentation and implementation for external API integration
- **Database Migration**: Step-by-step migration documentation with AI agent implementation
- **Microservice Creation**: End-to-end example of creating a new microservice

#### Domain-Specific Examples
- **Machine Learning Projects**: AI-ready documentation for ML workflows
- **DevOps Automation**: Infrastructure and deployment documentation
- **Mobile App Features**: Mobile-specific implementation patterns
- **Data Pipeline**: Analytics and data processing documentation

## Example Structure

Each example includes:
- **Original requirement**: The starting point
- **AI-ready documentation**: The enhanced documentation using our templates
- **Implementation result**: What the AI agent produced
- **Lessons learned**: Key insights and improvements

## Contributing Examples

To contribute a new example:
1. Ensure you have permission to share the content
2. Anonymize any sensitive information
3. Include the complete documentation → implementation → result cycle
4. Document lessons learned and recommendations
5. Follow the established example format

## Using Examples

Examples are most valuable when:
- Starting with a similar use case
- Learning documentation patterns
- Training AI agents on expected formats
- Validating new template approaches

## Example Format

```
example-name/
├── README.md                 # Example overview and context
├── original-requirement.md   # Starting point
├── ai-ready-docs/           # Enhanced documentation
│   ├── ticket.md
│   ├── architecture.md
│   └── implementation-plan.md
├── implementation/          # AI agent output
│   └── [generated files]
└── analysis.md             # Results and lessons learned
```
