# Ticket Templates

This directory contains templates for creating AI-ready tickets and issues that provide optimal context for autonomous implementation.

## Available Templates

### [AI-Ready Ticket Template](AI_READY_TICKET_TEMPLATE.md)
A comprehensive template for creating tickets that provide rich context for AI coding agents, including:
- System architecture context
- Implementation guidance
- Progressive testing strategies
- Validation checkpoints

## Usage

1. Copy the appropriate template
2. Fill in the specific details for your use case
3. Ensure all context sections are complete
4. Validate that the template provides sufficient information for autonomous implementation

## Template Guidelines

- **Be Specific**: Provide concrete examples and detailed context
- **Include Architecture**: Always include system architecture diagrams and component relationships
- **Define Success**: Clear, measurable success criteria
- **Progressive Validation**: Step-by-step verification strategies
